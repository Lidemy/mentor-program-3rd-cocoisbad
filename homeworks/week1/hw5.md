## 請解釋後端與前端的差異。
在畫面上可以看到的就是前端的部分
像是首頁
拿到網頁設計師的設計稿後
用 HTML CSS JS 把畫面做出來
即為前端的工作

後端為看不見得部分
像是前端送出一個關鍵字的搜尋
搜尋的過程就是後端的工作 
怎麼找到結果
最後在把結果送回前端

要求尋找有關 JavaScrit 的資料
Sever 收到後開始搜尋 database
並將結果 Response 給瀏覽器
